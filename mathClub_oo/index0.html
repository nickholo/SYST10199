  • Guess a Number
  • Convert Temperature

Increment a Number:

Allows the user to increment a number by 1

Add Two Numbers:

Adds two numbers together and displays the result together with the two values provided by the user

Guess a Number:

Allows the user to guess a number between 1 and 10 that is randomly chosen by the program

Convert Temperature:

Calculates a given between Celsius and Fahrenheit and displays both values

Fibonacci Sequence:

Calculates and displays a given number of Fibonacci numbers

Area of a Circle:

Calculates and displays the area of a circle given the radius
